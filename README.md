# Useful Tools

A suite of accessible, intuitive calculators and utilities for energy management, financial planning, and customer interaction scoring.

Built with Next.js 16, React 19, Tailwind CSS v4, and shadcn/ui.

## Development

```bash
npm install
npm run dev
```

The app runs on [http://localhost:9002](http://localhost:9002).

## Docker

```bash
docker compose up --build
```

## Scripts

| Command            | Description                        |
| ------------------ | ---------------------------------- |
| `npm run dev`      | Start development server (port 9002) |
| `npm run build`    | Production build                   |
| `npm start`        | Start production server            |
| `npm run lint`     | Run ESLint                         |
| `npm run typecheck` | Run TypeScript type checking      |
